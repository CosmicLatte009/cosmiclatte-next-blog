### 문제

페이지를 이동할 때마다 스크롤이 맨 위로 오게끔 하고 싶다.

### 해결 방법

useEffect 훅 내에서 `window.scrollTo(0,0)`을 작성해준다.

```JavaScript
useEffect(() => {
	window.scrollTo(0, 0);
}, []);
```
